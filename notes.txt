//! 3.Read about the difference between a window,screen and document in javascript

1. Window:
   - The window object represents the browser window or a frame.
   - It is the top-level object and acts as a global object for the browser.
   - Properties and methods related to the browser window are accessed through this object.

2. Screen:
   - The screen object provides information about the user's screen, such as its width, height, and color depth.
   - It is not a global object like window, but it is accessible as window.screen.

3. Document:
   - The document object represents the HTML document loaded in the browser.
   - It provides methods and properties to manipulate and interact with the content of the document, such as selecting elements or modifying their attributes.
